This plugin makes it possbile to use encrypted websites (https://) with the object Virtual HTTP input of the Loxone Config.

It calls the website in background via a https-connection and presents the content under the LoxBerry Plugin-Address via HTTP-Protocol for further treatment.
This Plugin has no settings.
